## Publicar grátis com GitHub Pages

1. Crie um repositório no GitHub.
2. Conecte este projeto ao repositório remoto:

```bash
git remote add origin https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
```

3. Faça o primeiro commit e envie:

```bash
git add .
git commit -m "feat: estrutura inicial da landing page"
git branch -M main
git push -u origin main
```

4. No GitHub, abra `Settings > Pages`.
5. Em `Build and deployment`, selecione:
   `Source: Deploy from a branch`
6. Escolha:
   `Branch: main`
   `Folder: / (root)`

Depois disso, o site ficará disponível na URL do GitHub Pages do repositório.
